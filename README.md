# MyAccount Automation

Automation toolkit for Brown University's MyAccount administrative workflows.

## Features

- User search
- Identity extraction
- Admin ID discovery
- Confirmation sheet generation
- Confirmation validation
- Add Admin IDs
- Revoke Admin IDs
- Purge Admin IDs
- History logging

## Requirements

- Python 3.12
- Chrome
- ChromeDriver

## Run

```bash
python main.py
